Builds the Cupertino button widget.

The button will display the child widget if provided. Otherwise, it will display a Text widget with text or an Icon widget with icon. If all are null, the button will be an empty SizedBox.

The onPressed callback is called when the button is tapped, and HapticFeedback.vibrate is called to provide tactile feedback.

Implementation

@override
Widget build(BuildContext context) {
  return CupertinoButton(
    padding: usePadding ? EdgeInsets.all(padding) : EdgeInsets.zero,
    onPressed: () {
      HapticFeedback.vibrate();
      onPressed();
    },
    child:
        child ??
        (text != null ? Text(text!) : icon ?? const SizedBox.shrink()),
  );
}
